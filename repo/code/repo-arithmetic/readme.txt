Title: Arithmetic
Category: Algorithm
Source: [VSI'08](http://web.cse.ohio-state.edu/~weide/rsrg/documents/2008/08WSHABBHKF.pdf)
Abstract: Build arithmetic operations based on the increment operation.
Description:
An implementation of addition using increments, multiplication using additions, and division using subtractions.
All operations are implemented with both iterative and recursive algorithms.
