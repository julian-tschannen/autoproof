Title: Doubly-Linked-List
Category: Data Structure
Source: [ECOOP'04](http://research.microsoft.com/en-us/um/people/leino/papers/krml132.pdf) / [SC'14](http://se.inf.ethz.ch/people/polikarpova/sc/)
Abstract: Implementation of a linked list with links to left and right neighbours.
Description:
A node in a (circular) doubly-linked list with a structural invariant that its left and right links are consistent with its neighbours.
