Title: Iterator
Category: Design Pattern
Source: [SAVCBS'06](http://www.eecs.ucf.edu/~leavens/SAVCBS/2006/challenge.shtml) / [SC'14](http://se.inf.ethz.ch/people/polikarpova/sc/)
Abstract: Usage of multiple iterators over a mutable collection.
Description:
Collection (arrayed list) that owns an array of data.
Iterators can be created over the collection, their validity (being in bounds) depends on the collection not being modified.
